Classifies comments as spam/ham using OpenRouter’s Llama 3.2 11B Free. Includes settings, logs, and bulk recheck.

== Installation ==
1. Upload the ZIP via Plugins → Add New → Upload.
2. Activate, then go to Settings → Silver Duck and set your OpenRouter API key.
