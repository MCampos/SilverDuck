Classifies comments as spam/ham using OpenRouter’s Llama models. Includes settings, logs, heuristics (links/blacklists), author field checks (name/email/url), optional blog-post context for relevance, and rate-limit backoff.

== Installation ==
1. Upload the ZIP via Plugins → Add New → Upload.
2. Activate, then go to Settings → Silver Duck and set your OpenRouter API key.
